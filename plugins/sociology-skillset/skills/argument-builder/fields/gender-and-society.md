# Field Profile: Gender & Society (Argument Builder)

This field profile adjusts argument-builder guidance for Theory sections targeting gender studies venues. Based on argument-style coding of 39 articles from *Gender & Society* (2024–2026), compared to the generalist corpus (n=80, *Social Problems* and *Social Forces*).

## When to Apply This Profile

Apply when the target venue or subfield is feminist sociology or gender studies — *Gender & Society*, *Signs*, *Feminist Studies*, *Journal of Marriage and Family* (gender-focused), or any article whose primary contribution is to feminist or gender theory.

Also apply when the article's core topic involves gender inequality, sexuality, masculinity/femininity, reproductive politics, or intersectional analysis of gender — even if targeting a generalist journal — as G&S conventions shape reader expectations.

## Same Toolkit, Different Settings

G&S theory sections use the same arc structures, paragraph functions, and turn moves as generalist sociology. The differences are distributional: which arcs and modes get used most often, how theory sections relate to case material, and how turns are constructed. No fundamentally novel argument structure appears — but the mix is distinctive enough that generalist defaults would steer authors wrong on mode selection and turn construction.

**A note on measurement**: These findings derive from qualitative coding of argument style (arc structure, turn type, theory-case integration, dominant mode) rather than word counts or citation statistics. Quantitative extraction of theory sections across corpora is sensitive to different markdown formats, heading conventions, and boundary definitions — we focus on what's robust to those artifacts.

## How G&S Differs from Generalist

Four key differences in argument style:

| Finding | Generalist | G&S | Implication |
|---------|-----------|-----|-------------|
| 1. Conceptual development is the dominant mode | Mixed modes | **51% conceptual development** | Theory sections should build new concepts, not merely review literature |
| 2. Integration-promise turns are common | Gap statements dominate | **28% integration-promise** | "By bringing X and Y into conversation..." is a strong, recognized move |
| 3. Building-blocks is the dominant arc | Mixed (Funnel common) | **Building-blocks 77%** | Assemble concepts sequentially; funnel is a secondary option, not the default |
| 4. Theory-case integration is distributed | Theory section is "pure theory" | **Separate 36%, Woven 33%, Theory-first 31%** | All three patterns are legitimate; choose based on how case material serves the argument |

**Root pattern**: G&S theory sections build conceptual architecture. The modal article assembles theoretical building blocks — often from feminist theory, intersectionality, and adjacent traditions — to construct or extend a concept, arriving at the contribution through a gap statement or an integration promise. The theory section is doing *conceptual work*, not merely surveying a literature.

## Adjusted Guidance

### Arc Structure (G&S)

| Arc | Generalist | G&S | When to use |
|-----|-----------|-----|-------------|
| Building-blocks | Available | **77% (dominant)** | Default choice. Concept A → Concept B → Concept C → synthesis. Natural for G&S's multi-tradition conceptual work. |
| Funnel | Common default | **13%** | Broad context → narrowing. Works for gap-filler contributions, but not the default. |
| Dialogue | Available | **8%** | Position A vs Position B → resolution. Rare but legitimate for articles adjudicating theoretical debates (e.g., co-optation vs. reframing). |
| Problem-response | Available | **3%** | When existing conceptual frameworks demonstrably fail. Very rare in G&S. |

### Theory-Case Integration (G&S)

This is more evenly distributed in G&S than in either the generalist or social movements corpus. All three main patterns appear in roughly equal proportions — the choice depends on the article's relationship between theory and case.

| Pattern | G&S % | Description | When to use |
|---------|-------|-------------|-------------|
| Separate | **36%** | Distinct "background" and "theory" subsections | When extensive case context is needed but shouldn't interrupt theoretical flow; common with non-U.S. cases requiring contextual framing |
| Woven | **33%** | Case details appear throughout theory discussion | When the case illuminates theoretical points; especially effective for concept-building articles where case and theory are co-constitutive |
| Theory-first | **31%** | Pure theory precedes any case mention | Default for quantitative articles and densely conceptual contributions; keeps theory clean |
| Case-first | **0%** | Case background precedes theory | Not observed in this corpus; avoid |

**Practical implication**: Choose based on how your case relates to your theory. If working with a non-U.S./non-Western case, the separate pattern (context subsection + theory subsection) is well-established. If building a new concept, weaving case throughout can show how theory and evidence are co-constitutive. If the contribution is purely conceptual, theory-first keeps the argument clean.

### The Turn (G&S)

| Turn type | G&S % | Description | Signature phrase |
|-----------|-------|-------------|-----------------|
| Gap statement | **44%** | Identifies what's missing | "What remains unclear is..." / "Few studies have examined..." |
| Integration promise | **28%** | Brings traditions together | "By bringing X and Y into conversation..." / "We integrate... with..." |
| Extension claim | **18%** | Extends existing framework | "We extend X by..." / "This framework can be applied to..." |
| Limitation critique | **10%** | Challenges existing approach | "This framework cannot account for..." / "The dominant approach overlooks..." |

**Key contrast with generalist**: Integration-promise turns are notably common in G&S (28% vs. rare in generalist). This reflects G&S's orientation toward synthesizing across feminist theory, intersectionality, and adjacent traditions to build new conceptual vocabularies. If your contribution brings two or more theoretical traditions into productive conversation, the integration-promise turn is a strong choice.

**Key contrast with social movements**: Limitation-critique turns are less common in G&S (10% vs. 24% in SM). G&S theory sections tend to build on existing feminist frameworks rather than dismantling them.

### Dominant Mode (G&S)

| Mode | G&S % | What the theory section is primarily doing |
|------|-------|------------------------------------------|
| Conceptual development | **51%** | Building new concepts, typologies, or analytical vocabularies |
| Framework application | **21%** | Explaining a theory to be applied to a new case or context |
| Literature review | **18%** | Surveying what's known across relevant traditions |
| Hypothesis derivation | **10%** | Building to testable predictions (quantitative articles) |
| Debate mapping | **0%** | Laying out competing positions |

Conceptual development is the dominant mode by a wide margin — over half of G&S theory sections are primarily building new concepts. This is the single most distinctive feature of G&S argument style. Approximately 38% of articles in the corpus introduce a new named concept (e.g., "carceral care economy," "diasporic capital," "spatialized feelings," "feminist erosion," "insurgent reproductive citizenship").

**Practical implication**: If targeting G&S, your theory section should be doing conceptual work — not merely reviewing literature. Even framework-application articles in G&S tend to extend or modify the framework they apply. Pure literature surveys without conceptual contribution are relatively rare (18%).

### Hypotheses (G&S)

| | G&S % | Notes |
|---|-------|-------|
| No formal hypotheses | **87%** | All qualitative and most theoretical articles |
| Numbered hypotheses (H1, H2...) | **10%** | All quantitative articles |
| Informal expectations | **3%** | Research questions rather than hypotheses |

**Practical implication**: G&S is overwhelmingly qualitative (87% of corpus). Formal hypotheses are appropriate only for quantitative articles. Qualitative articles state research questions or conceptual aims, never formal hypotheses.

### Opening Moves (G&S)

| Opening move | G&S % | Description |
|-------------|-------|-------------|
| Concept-led | **31%** | Opens by defining or introducing the central concept |
| Phenomenon-led | **23%** | Opens with the social phenomenon motivating the study |
| Theory-led | **21%** | Opens with a named theorist or theoretical tradition |
| Case-led | **13%** | Opens with the empirical case or site |
| Debate-led | **13%** | Opens by mapping a contested theoretical terrain |

**Practical implication**: Concept-led openings are the most common, consistent with G&S's orientation toward conceptual development. Lead with the concept your theory section is building toward. Phenomenon-led and theory-led openings are also well-represented. Case-led openings work well when a non-U.S. context needs to be established before theory can be developed.

### Subsection Structure (G&S)

| Metric | Value |
|--------|-------|
| Median subsections | **3** |
| Mean subsections | **2.8** |
| Range | **1–5** |
| Heading style | **62% substantive, 36% mixed, 3% generic** |

**Practical implication**: Use 2–3 subsections with substantive headings that name theoretical content (e.g., "Gendered Temporalities," "Care Work at the Intersections") rather than generic labels ("Literature Review," "Background"). When using the separate theory-case pattern, one subsection is typically a contextual heading naming the case or site.

## G&S-Specific Patterns

Three patterns appear with high frequency in G&S theory sections that are less prominent in the generalist corpus:

### 1. Intersectionality as Organizational Logic (~87%)

Intersectionality is not merely cited — it organizes how theory sections are structured. Multiple axes of inequality (race, class, gender, sexuality, nationality, age) are woven into the theoretical argument rather than treated as control variables or secondary considerations. Theory sections that address gender in isolation without intersectional framing are rare.

**Implication**: Build intersectionality into the structure of your theory section, not as an add-on paragraph. Each theoretical building block should address how the concept operates across intersecting axes.

### 2. Feminist Theory as Foundational (~97%)

Nearly every article engages feminist theory as a foundational framework, not as one tradition among many. Common anchoring theorists include Collins, Connell, hooks, Butler, Crenshaw, Hochschild, and Ahmed. Theory sections assume fluency with core feminist concepts (doing gender, hegemonic masculinity, compulsory heterosexuality, controlling images, care work).

**Implication**: Don't explain basic feminist concepts. Assume your reader knows them. Build from these foundations toward your specific contribution.

### 3. Global/Transnational Scope (~62%)

A majority of articles in the corpus study non-U.S. contexts or adopt explicitly transnational framing. This is notably higher than generalist sociology journals. Theory sections frequently engage with postcolonial, decolonial, or Global South feminist scholarship alongside Western feminist theory.

**Implication**: When working with a non-U.S. case, the theory section should engage with locally grounded feminist scholarship — not merely apply Western frameworks to non-Western contexts. The separate theory-case integration pattern (36%) often serves this purpose, providing contextual grounding before or alongside theoretical framing.

## Cluster Interactions

The contribution-type clusters (Gap-Filler, Theory-Extender, etc.) still apply. Here's how they interact with G&S-specific patterns:

| Contribution cluster | Recommended G&S arc | Likely turn type | Theory-case pattern | Likely mode |
|---------------------|---------------------|-----------------|-------------------|-------------|
| Gap-Filler | Building-blocks or Funnel | Gap statement | Separate or Theory-first | Literature review |
| Theory-Extender | Building-blocks | Extension claim or Integration promise | Theory-first or Woven | Conceptual development |
| Concept-Builder | Building-blocks | Integration promise or Gap statement | Woven | Conceptual development |
| Synthesis Integrator | Building-blocks | Integration promise | Woven or Separate | Conceptual development |
| Problem-Driven | Building-blocks | Gap statement or Extension claim | Separate | Framework application |

**G&S-specific note**: The Concept-Builder cluster is disproportionately represented in G&S (51% of articles are primarily doing conceptual development). When in doubt about contribution type, lean toward Concept-Builder or Theory-Extender rather than Gap-Filler — G&S rewards conceptual ambition.

## Writing Checklist

### Before Finalizing a G&S Theory Section

Verify:

- [ ] **Arc structure**: Is the arc appropriate? Building-blocks is the G&S default (77%). Funnel is available but secondary. Dialogue and problem-response are rare — use only with clear justification.
- [ ] **Conceptual work**: Is the theory section doing conceptual development, not merely reviewing literature? G&S theory sections build toward something — a new concept, a synthesis, an extension — not just a survey.
- [ ] **The turn**: Is the gap/contribution pivot specific? If using an integration-promise turn, are both traditions clearly developed before the synthesis move?
- [ ] **Turn type**: Does the turn match the contribution? Integration-promise turns are strong for G&S; limitation-critique turns are less common than in other subfields.
- [ ] **Intersectionality**: Is intersectional analysis woven into the theoretical architecture, not added as an afterthought paragraph?
- [ ] **Feminist theory fluency**: Does the theory section assume familiarity with core feminist concepts rather than explaining them from scratch?
- [ ] **Theory-case integration**: If case material appears in the theory section, does each reference advance the theoretical argument? If using separate subsections, does the contextual subsection do analytical work (not just describe a setting)?
- [ ] **Subsection headings**: Are headings substantive (naming theoretical content) rather than generic ("Literature Review")? G&S theory sections use substantive labels (62%).
- [ ] **Hypotheses**: If qualitative, are formal hypotheses absent? If quantitative, are hypotheses numbered and derived from theory?
- [ ] **Global framing**: If studying a non-U.S. context, does the theory section engage with locally grounded scholarship rather than only applying Western frameworks?
